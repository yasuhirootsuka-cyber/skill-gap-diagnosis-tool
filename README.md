# スキルギャップ診断ツール | 株式会社コトラ

金融・コンサル・IT業界専門のスキルギャップ診断ツールです。現在のスキルと目標ポジションに必要なスキルのギャップを可視化し、具体的なスキルアップの方向性を提案します。

## 🎯 プロジェクト概要

このツールは、ユーザーの現在のスキルレベルと目標ポジションで求められるスキルレベルを比較し、ギャップを可視化します。診断結果は、レーダーチャートで表示され、優先的に伸ばすべきスキル、おすすめの学習方法、マッチする求人情報を提供します。

## ✨ 主な機能

### 現在実装済みの機能

1. **スキルギャップ診断フロー**
   - 3つの質問によるインタラクティブな診断
   - キャリアパス選択（金融・コンサル・IT・経営幹部）
   - 現在のスキルと目標スキルの5段階評価
   - **📊 レベル評価基準ガイド表示**（各レベルの具体的な説明）
   - **💡 スキル別評価基準の詳細表示**（?ボタンで展開）
   - **ホバーヒント機能**（ボタンにカーソルを合わせると具体例を表示）
   - プログレスバーでの進捗表示

2. **スキルギャップ分析**
   - 業界別の5つのスキルカテゴリー評価
   - 現在スキルと目標スキルの差分計算
   - スキルレベルの判定（上級・中級・初級）
   - ギャップの大きいスキルの特定

3. **詳細な診断結果**
   - レーダーチャートによる可視化
   - スキル別のギャップ分析（バーチャート）
   - 優先的に伸ばすべきスキルのリスト
   - 各スキルのギャップ値表示

4. **パーソナライズドアドバイス**
   - キャリアパス別の学習リソース提案
   - 優先度に基づく学習計画
   - 具体的な資格・講座の推奨
   - 実務経験の積み方アドバイス

5. **求人マッチング**
   - スキルレベルに応じた求人紹介
   - キャリアパス別の求人情報
   - 求人の詳細説明と必要スキルレベル

6. **データ保存機能**
   - RESTful Table APIによる診断結果の保存
   - メールアドレスと診断データの紐付け
   - 後続のマーケティング活用が可能

## 📊 データモデル

### skill_gap_diagnostics テーブル

| フィールド名 | 型 | 説明 |
|------------|-----|------|
| id | text | レコードID |
| email | text | ユーザーのメールアドレス |
| career_path | text | キャリアパス（finance/consulting/it/general） |
| current_skills | text | 現在のスキル評価（JSON形式） |
| target_skills | text | 目標スキルレベル（JSON形式） |
| avg_current | number | 現在スキルの平均 |
| avg_target | number | 目標スキルの平均 |
| avg_gap | number | スキルギャップの平均 |
| diagnostic_date | datetime | 診断実施日時 |

## 🚀 使い方

### 1. 診断の開始
トップページの「診断を開始する」ボタンをクリック

### 2. 質問への回答（全3問）
- **Q1: キャリアの方向性選択**
  - 金融スペシャリスト
  - コンサルタント
  - IT・エンジニア
  - 事業会社の経営幹部

- **Q2: 現在のスキル評価**
  - 選択したキャリアパスに応じた5つのスキルを5段階評価
  - 例）金融：財務分析、バリュエーション、会計知識、リスク管理、金融商品知識

- **Q3: 目標スキルレベル**
  - 同じ5つのスキルについて目標レベルを5段階評価

### 3. プレビュー結果の確認
総合スキルレベル（上級・中級・初級）と**キャリアパス別のパーソナライズドメッセージ**を表示

#### 📊 診断完了メッセージパターン（36パターン）

**キャリアパス × スキルレベル × ギャップ評価**の組み合わせで、計36パターンのメッセージを用意：

| キャリアパス | スキルレベル | ギャップ評価 | メッセージ例 |
|------------|------------|------------|------------|
| 金融スペシャリスト | 上級 | 優秀 | 「素晴らしい！金融スペシャリストとして即戦力レベルのスキルをお持ちです」 |
| 金融スペシャリスト | 中級 | 良好 | 「金融スキルの基礎は固まっています。実践経験を積む段階です」 |
| 金融スペシャリスト | 初級 | 要努力 | 「金融知識の習得から始めましょう。資格取得も視野に入れると良いでしょう」 |
| コンサルタント | 上級 | 優秀 | 「素晴らしい！コンサルタントとしてトップレベルのスキルを持っています」 |
| IT・エンジニア | 中級 | 良好 | 「IT技術の基礎は固まっています。実装経験を積む段階です」 |
| 経営幹部 | 上級 | 優秀 | 「素晴らしい！経営幹部として十分なスキルとリーダーシップをお持ちです」 |

**スキルレベル判定基準：**
- **上級**：平均4.0以上
- **中級**：平均3.0〜3.9
- **初級**：平均3.0未満

**ギャップ評価基準：**
- **優秀**（excellent）：ギャップ0.5以下
- **良好**（good）：ギャップ0.5〜1.5
- **要努力**（almostThere）：ギャップ1.5超

### 4. メールアドレス登録
詳細レポート受け取りのためのメール登録

### 5. 詳細分析レポートの閲覧

#### **📊 8つのセクションで構成された総合レポート**

1. **診断サマリー**
   - キャリアパス、現在・目標スキルレベル、平均ギャップ
   - パーソナライズドメッセージ

2. **スキル評価チャート**
   - レーダーチャート（現在 vs 目標）
   - 視覚的なスキル比較

3. **スキル別詳細分析**
   - 各スキルのギャップをバーチャートで表示
   - 具体的な数値とギャップレベル

4. **優先的に伸ばすべきスキル TOP3**
   - ギャップと優先度スコアで順位付け
   - 集中すべきスキルが明確

5. **スキル別学習ロードマップ**
   - 各スキルの学習ステップをタイムライン表示
   - ギャップに応じた学習期間の目安
   - 具体的な学習リソース提案

6. **推奨学習リソース**
   - オンライン講座、実務経験、資格取得など
   - 5つの具体的な学習方法

7. **キャリアアドバイス**
   - キャリアパス別の具体的アドバイス
   - スキルレベルに応じた提案

8. **次のステップ**
   - ギャップに応じた具体的なアクションプラン
   - 3〜5つの実行可能なステップ

#### **🖨️ 印刷機能**
- ワンクリックで印刷可能
- A4サイズに最適化されたレイアウト
- 面接や上司への報告に活用可能

#### **📥 PDFダウンロード機能（NEW!）**
- **ワンクリックでPDF生成・ダウンロード**
- レポート内容を完全にPDF化
  - 診断サマリー
  - スキル詳細（数値データ）
  - レーダーチャート（画像として埋め込み）
- 進捗表示付き（生成中のローディング画面）
- ファイル名自動生成：`SkillGapReport_[キャリア]_[日付].pdf`
- オフライン閲覧・メール共有に最適

## 🎨 UI改善内容（2024-12-16）

### レベル評価の明確化
ユーザーがスキルレベルを評価しやすいように、以下の機能を追加しました：

1. **📊 レベル評価基準ガイド**
   - スキル評価画面の上部に常に表示
   - Lv1〜Lv5の各レベルの意味を明確に説明
   - 視覚的にわかりやすいデザイン

2. **💡 スキル別評価基準**
   - 各スキル名の横に「?」ボタンを配置
   - クリックするとそのスキルの具体的な評価基準を表示
   - キャリアパス別に最適化された基準例
   - 例）プログラミング：
     - Lv1: 基本的な文法を理解している
     - Lv3: 実務で品質の高いコードを書ける
     - Lv5: 技術コミュニティをリードし、業界標準を作れる

3. **ツールチップ表示**
   - 評価ボタンにカーソルを合わせると、そのレベルの具体例を表示
   - マウス操作でスムーズに確認可能

### 効果
- ユーザーが自信を持ってスキルレベルを評価できる
- 評価基準が統一され、より正確な診断結果を得られる
